# Ejercicios de Lógica en JavaScript

Este repositorio contiene una serie de ejercicios diseñados para mejorar tus habilidades de lógica de programación utilizando JavaScript.

## Estructura de carpetas

Cada ejercicio se encuentra en su propia carpeta, con la siguiente estructura:

- `ejercicio-01/`
  - `index.html`: Instrucciones del problema.
  - `ejercicio-01.js`: Solución en JavaScript.
- `ejercicio-02/`
  - ...
- ...
